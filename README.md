## Running the app

Clone the repository, install dependencies, and start the dev server:

```bash
git clone https://github.com/AnkhEl/scrums-react.git
cd scrums-react
npm install
npm run start
```

Open the app at http://localhost:1234 (Parcel default). Optional commands:

```bash
# Run TypeScript type-check
npm run typecheck

# Build for production
npm run build
```
